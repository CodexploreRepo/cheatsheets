# iTerm Shortcut
- `CTRL + L`: to clear the terminal
# Open new terminals
- `CMD + D`: to split the terminal into more sessions
  - `SHIFT + CMD + D`: to split the terminal into more sessions horizontally
- `CMD + W`: to close the terminal sessions
